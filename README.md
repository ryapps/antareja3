<h2>🏃‍♂️ How to run</h2>

Clone this repository:

```bash
git clone https://github.com/mokletdev/antareja.git
```

Install dependencies

```bash
npm install
```

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
